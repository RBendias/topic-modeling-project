# Topic Modeling Project - Challenge: 7.1 Automated Stakeholder Analysis for Hedera

I&E challenge Market analysis using IT tools - team 7.1

# Setup

Install pipenv 

```bash
$ pip install pipenv
```

Install all dependencies from Pipfile

```bash
$ pipenv install
```

Activate pipenv shell:

```bash
$ pipenv shell
```

Start Jupyter Notebook in shell
```bash
$ jupyter notebook
```

Happy Coding.

# Updating Pipenv
If you add any requirements make sure to add them to the pipenv file
Installing new packages via: 
```
pipenv install <package_name>
```
